# **一.  缘由**

---

日期转化为时间戳，安卓上正常，ios 出现 NAN

# **二.  解决**

---

分析后发现这是因为 ios 对于时间格式为 “YY-MM-DD 00：00：00” 这种 ‘-’ 格式的不能转换为时间戳，

只需要将其转换为 ‘YY/MM/DD 00：00：00’ 这类 ‘/’ 格式的时间。

以下是转换代码：

```js
var date_a = '2020-8-5 09:23:20'
// 转换为'/'格式
var date_b = date_a.replace(/-/g,'/')
// 转换为时间戳
// 1.   Date.parse(date_b)
// 2.   new Date(date_b).getTime()
```
