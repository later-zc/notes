# 一. 常用快捷键

---

```js
Shift + Ctrl + c // 当前编辑器窗口打开cmd
alt + upArrow // 向上移动当前行
alt + downArrow // 向下移动当前行
shift + alt + upArrow // 向上复制当前行
shift + alt + downsArrow // 向下复制当前行
ctrl + k + x // 裁剪尾随空格
tab // 当处于一个字符尾端时，可生成对应的组件形式的字符
// 如：abb => <abb></abb>N
ctrl + enter // 在下面插入行
ctrl + shift + enter // 在上面插入行
```





# 二. 自定义快捷键

---

```js
ctrl + ; // 指针跳转到当前行的行尾
ctrl + '' // 指针跳转到当前行的行首
ctrl + k // 删除行
```
