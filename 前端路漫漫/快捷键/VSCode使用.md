# 一. 常用快捷键

---

```js
Shift + Ctrl + c // 当前编辑器窗口打开cmd
Ctrl + Shift + `` // 创建一个新终端
Ctrl + `` // 切换终端面板
Ctrl + B // 切换侧边栏

ctrl + k + x // 裁剪尾随空格
Ctrl + home/end // 导航到文件的开头/结尾

alt + upArrow // 向上移动当前行
alt + downArrow // 向下移动当前行

shift + alt + upArrow // 向上复制当前行
shift + alt + downsArrow // 向下复制当前行

ctrl + enter // 在下面插入行
ctrl + shift + enter // 在上面插入行

Ctrl + Tab // 切换标签页

// template
tab // 当处于一个字符尾端时，可生成对应的组件形式的字符
// 如：abb => <abb></abb>

// vscode集成终端
tab // 当输入当前目录下某个文件的文件名前几个字符时，可快速补全完整文件名
cls // 清除命令行历史记录
Ctrl + C // 退出终端正在运行的当前进程

Home // 跳到当前代码行首
end // 跳到当前代码行尾
```





# 二. 自定义快捷键

---

```js
ctrl + ; // 指针跳转到当前行的行尾
ctrl + '' // 指针跳转到当前行的行首
ctrl + k // 删除行
```
