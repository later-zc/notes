# 一. 通用快捷键

---

```js
Ctrl + A  // 全选
Ctrl + C  // 复制
Ctrl + X  // 剪切
Ctrl + V  // 粘贴
Ctrl + Z  // 撤消
Ctrl + Y  // 恢复撤销内容
Ctrl + F  // 寻找
Ctrl + S  // 保存
```





# 二. 常用快捷键

---

```js
F2  // 文件夹重命名
Win + R  // 调用运行对话框（输入cmd回车可以打开终端窗口）
Win + D  // 显示/恢复桌面
Win + L  // 锁定电脑
Win + E  // 打开文件管理器/我的电脑
Win + X  // 系统设置菜单
win + I // 系统设置
PrtSc // 屏幕快照
Alt + Tab  // 快速切换多窗口

shift + 24个字母 // 输入的字面为大写形式，且无需切换中英文  

Win + Tab  // 显示你打开的所有窗口，你要哪个点哪个
Win + T  // 任务栏上固定的应用之间相互切换
Alt + F4  // 关闭窗口

Ctrl + Shift + Esc  // 打开任务管理器
Ctrl + Alt + Del // 打开快速重启

ctrl + . // 这会让你的符号全部变成英文的, 再按又变回中文
shift // 等价于上作用

win + ⬆ // 放大窗口至全屏
win + ⬇ // 缩小窗口
```
