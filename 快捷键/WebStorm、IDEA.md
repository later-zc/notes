## 01. 查找替换

- 在文件中查找：`Ctrl + Shift + F`
- 在文件中替换：`Ctrl + Shift + R`
- 在当前文件中查找：`Ctrl + F`
- 在当前文件中替换：`Ctrl + R`
- 将下一个查找匹配项添加到选择：`Alt + J`
- 扩展选择：`Ctrl + W`
- 搜索任何东西：`shift + shift`



## 02. 跳转操作

- 转到文件：`Ctrl + Shift + N`
- 打开最近的项目：手工操作（自定义：`Alt + E`）
- 打开最近的文件：`Ctrl + E`
- 返回：`Ctrl + Alt + LeftArrow`
- 前进：`Ctrl + Alt + RightArrow`
- 聚焦到第一个单元格：`Ctrl + Home`
- 聚焦到最后一个单元格：`Ctrl + End`
- 跳转最近使用的标签页：`Ctrl + Tab`
- 查看最近的位置：`Ctrl + Shift + E`



## 03. 行操作

- 向下复制行：`Ctrl + D`
- 向上/下移动行：`Alt + Shift + UpArrow/DownArrow`
- 聚集到行首：`Home`
- 聚集到行尾：`End`
- 在上方插入行：`Ctrl + Alt + Enter`
- 在下方插入行：`shift + Enter`



## 04. 面板操作

- 查看/隐藏终端面板：`Alt + F12`
- 关闭当前编辑器标签页：`Ctrl + F4`
- 打开设置页：`Ctrl + Alt + S`



## 05. 窗口操作

- 新建窗口：`Alt + F6`
- 关闭当前窗口：`Alt + F4`（自定义：`Ctrl + Shift + W`）



## 06. 其他操作

- 快速生成类的属性的 getter 和 setter 方法：`Alt + Insert`
